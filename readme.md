# TAKIM 37

Datathon project for team 37.

## Files

- `YZTA_Datathon_1.ipynb` and `YZTA_Datathon_2.ipynb`: analysis and modeling notebooks
- `train.csv` and `test_x.csv`: local data files
- `submission.csv`: sample output
- `research/`: notes, benchmarks, and strategy docs

## Setup

Keep the data files in the project root and run the notebooks in order.
